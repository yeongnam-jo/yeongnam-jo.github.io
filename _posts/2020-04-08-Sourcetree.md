### Sourcetree 기본 사용 방법 및 GitHub 연동 방법
1. Clone 버튼을 눌러 GitHub repository 파일을 로컬로 받아온다.
2. 브랜치 아이콘을 눌러 생성 및 삭제해본다.
3. 브랜치명을 더블클릭해 checkout 해본다.
4. commit, push, pull 등을 진행해본다.
